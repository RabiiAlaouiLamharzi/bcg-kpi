What This Means for You
Capacity risk
Data creation is outpacing storage growth by 4 points per year. A shortfall is likely by 2024. Start capacity planning conversations now.
Immediate attention needed
Invest in Object storage
Object storage is growing at 10% per year. File storage growth has turned negative. Redirecting budget toward Object storage is well-timed.
Strategic opportunity
Review cloud contracts
Public cloud now represents 41% of enterprise IT spend. Assessing whether a hybrid model could reduce costs is worth pursuing.
